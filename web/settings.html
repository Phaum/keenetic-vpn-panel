Настройки adguardvpn-cli

Transparent proxy через redsocks

Этот режим прозрачно перенаправляет TCP-трафик из выбранных LAN-подсетей в local redsocks, а тот уже отправляет соединения в listener adguardvpn-cli. Для UDP/DNS нужен отдельный слой. Если заданы destination CIDR или домены, редирект будет выполняться только для них; доменные правила требуют dnsmasq с поддержкой ipset. Для полноценного TCP+UDP используй режим TUN policy routing.
